# AI Virtual Doctor for preliminary health recommendations

An AI-powered healthcare assistance platform that helps users identify possible diseases based on symptoms, view recommended medicines and precautions, find doctors, and access adult care support resources.

---

## Project Overview

The AI Virtual Doctor for preliminary health recommendations  is a machine learning-based healthcare application designed to provide preliminary health guidance by analyzing user symptoms and predicting potential diseases.

The platform integrates:

- Disease Prediction using Machine Learning
- Medicine Recommendations
- Disease Precautions
- Adult Care Support Module

This project aims to provide quick health-related insights while encouraging users to consult healthcare professionals for accurate diagnosis and treatment.

---

## Features

### Disease Prediction

Users enter symptoms and the system predicts the most likely disease using a trained machine learning model.

### Medicine Recommendation

Displays medicines commonly associated with the predicted disease.

### Precaution Suggestions

Provides preventive measures and precautions for the predicted disease.

### Adult Care Module

Provides resources and support information related to adult care services.

## Installation


```bash

```

### Install Dependencies

pip install -r requirements.txt

## Run Application

```bash
streamlit run app.py
```
